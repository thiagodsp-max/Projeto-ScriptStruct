# Projeto-ScriptStruct
Projeto de Software, que objetiva a criação de um aplicativo apropriado para escritores e entusiastas por literatura, permitindo que o usuário possa gerir suas narrativas.
Dentre as Funcionalidades prometidas está:
- Criação de pastas e arquivos, para permitir o usuário organizar suas histórias
- Formatação de Texto, permitindo que o usuário revise o texto no próprio aplicativo
- Exportação da História, no formato pdf, permitindo o compartilhamento da obra após sua conclusão
- Configuração de Texto, customize o texto para deixar a leitura mais confortável, como fundo claro/escuro
- Status de História, permite que o usuário possa definir a prioridade de seus projetos
- Criação de Notas, com título e descrição, para escrever ideias, sem estarem vinculadas a um projeto
- Estrutura Assíncrona, o aplicativo não dependerá de Internet
